# Sistema de Gestão de Pedidos - O Moelas

## Como Executar
Este projeto será dividido em Backend (Python) e Frontend (Processing/p5.js). As instruções de instalação serão atualizadas à medida que o código for submetido.

## Tecnologias Escolhidas
* [cite_start]**Backend:** Python/Django - Escolhido pela robustez e por ser a stack recomendada para uma implementação sólida[cite: 32, 33].
* [cite_start]**Frontend:** Processing (p5.js) - Escolhido para criar uma interface intuitiva e funcional para o cliente e para a cozinha[cite: 40, 60].
* [cite_start]**Base de Dados:** SQLite - Utilizado pela simplicidade de configuração local[cite: 33, 136].

## Funcionalidades Base
* [cite_start]**Interface do Cliente:** Menu com 5 categorias (Entradas, Sopas, Carne, Peixe, Sobremesa)[cite: 41, 46].
* [cite_start]**Dashboard da Cozinha:** Quadro com colunas (Order Preview, Preparing, Cooling Down, Ready to Serve, Concluded)[cite: 61, 63].
* [cite_start]**Modelo de Dados:** Normalizado com Items, Orders e Order Lines[cite: 77, 79].
